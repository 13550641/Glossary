Canvas

Glossary

The Canvas Object is new in HTML5.

The HTML5 canvas tag is used to draw graphics, on the fly, with JavaScript. Access a Canvas Object You can access a canvas element by using getElementById().

The canvas element has no drawing skills of its own (which is just a graphics container) - you must use a script to actually draw the graphics.

The getContext () method returns an object that provides methods and properties to draw on the canvas.

This reference will cover the properties and methods of the getContext ("2d") object, which can be used to draw text, lines, rectangles, circles, and much more - on the canvas.
